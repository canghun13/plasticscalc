About the project

Practical reference tools for plastics processing

PlasticsCalc is a free, static reference site for people who need a clear first-pass calculation before moving to supplier data, process validation, or engineering review.

What the site covers

The initial focus is injection molding: machine sizing, shot planning, cooling and cycle estimates, output, material use, and basic costing. The structure can later support materials, extrusion, blow molding, thermoforming, and quality tools without diluting the injection molding focus.

Who it is for

Engineers, molders, technicians, estimators, buyers, students, and project teams can use the tools to organize assumptions and compare scenarios. The site does not provide accounts, paid calculations, supplier quotations, or production approvals.

How calculations are maintained

Each calculator states its formula, units, example, and limitations. We prefer manufacturer, material-supplier, association, and engineering references. When an input depends on a specific resin grade, mold geometry, or machine, the tool identifies that dependency rather than presenting a universal value.

What an estimate is not

A calculator result is not a machine specification, mold design release, resin recommendation, safety decision, or guaranteed cost. Critical decisions should be checked against the relevant machine documentation, material data sheet, validated process, simulation, and trial results.

Content updates

Tools and guides are expanded only when they add a distinct practical use case. We avoid mass-producing near-duplicate pages; corrections and source-based improvements are recorded in the project handover.
