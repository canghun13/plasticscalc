Decision map

Choose the next calculation from the missing evidence.

Each path begins with a different source of evidence and ends with a different cross-check. Use the result to narrow a decision, not to replace the next check.

Machine fit

Start with CAD projection and complete shot. Cross-check clamp force, tie-bar/platen fit, and injection-unit capacity.

Cycle and output

Start with a defined observed cycle. Cross-check cooling, automation, uptime, and good-part yield before planning capacity.

Material use

Start with exact grade density or weighed part mass. Cross-check solid versus melt state before using weight or volume.

Cost exposure

Start with material boundary and production life. Cross-check resin policy, scrap classification, conversion cost, and tooling scope.
