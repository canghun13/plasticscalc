Machine and mold screen

Four independent checks before a production assumption.

A capacity number is only credible when geometry, injection unit, thermal behavior, and production losses are each defined on their own basis.

01 / Geometry

Use CAD projection and the actual runner layout before estimating clamp force.

02 / Injection unit

Build complete shot mass, then compare volume and cushion with the documented screw and resin basis.

03 / Thermal cycle

Use the controlling wall section and ejection criterion; do not turn a wall model directly into a production setting.

04 / Good output

Apply uptime and yield separately when converting cycle time into capacity, cavities, or cost.

The linked guides explain the pressure, cooling, shot-capacity, and cost assumptions that need an external source or trial check.
