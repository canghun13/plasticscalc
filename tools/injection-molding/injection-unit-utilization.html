Injection Unit Utilization Calculator

Compare a planned shot volume with a documented usable injection-unit volume. Use this as a documented planning estimate, then validate it against the actual machine, mold, resin, and process.

Formula and method

Utilization = planned shot volume / usable injection volume x 100.

Use actual shot volume and the machine maker volume definition; retain the resin and temperature basis.

Inputs and units

Use controlled records, CAD, measured production data, the exact resin data sheet, or machine-maker specifications. Keep the material and time basis with the result.

Worked example

180 cm3 / 240 cm3 = 75%.

What the result supports

This result is a first-pass screen for injection unit planning. Use it to identify the next record or engineering check, not as a release decision.

Sanity check

Check the unit basis and whether all quantities describe the same cavity, shot, machine, or production window before comparing values.

Important limitation

A percentage is a screening value, not a machine-selection approval.

Reference: ENGEL: machine data example.
