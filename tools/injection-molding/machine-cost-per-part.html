Machine Cost Per Part Calculator

Allocate machine-hour cost to one molded part using cycle, cavities, and uptime. Use this as a documented planning estimate, then validate it against the actual machine, mold, resin, and process.

Formula and method

Machine cost = hourly rate x cycle / 3,600 / cavities / uptime.

Use an internal approved machine-hour rate and the same uptime definition used in capacity planning.

Inputs and units

Use controlled records, CAD, measured production data, the exact resin data sheet, or machine-maker specifications. Keep the material and time basis with the result.

Worked example

A $60/h machine, 30 s cycle, four cavities, 90% uptime gives $0.14 per part.

What the result supports

This result is a first-pass screen for costing planning. Use it to identify the next record or engineering check, not as a release decision.

Sanity check

Check the unit basis and whether all quantities describe the same cavity, shot, machine, or production window before comparing values.

Important limitation

This excludes material, labor, tooling, quality, and overhead unless your rate includes them.

Reference: ARBURG: cycle-time optimisation.
