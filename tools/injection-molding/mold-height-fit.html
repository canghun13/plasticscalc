Mold Height Compatibility Calculator

Check whether a mold height lies inside the machine’s stated height range. Use this as a documented planning estimate, then validate it against the actual machine, mold, resin, and process.

Formula and method

Margin = min(mold height - minimum, maximum - mold height).

Use the complete shut-height mold stack and the machine’s min/max mold-height specification.

Inputs and units

Use controlled records, CAD, measured production data, the exact resin data sheet, or machine-maker specifications. Keep the material and time basis with the result.

Worked example

A 450 mm mold in a 300–600 mm range has 150 mm limiting margin.

What the result supports

This result is a first-pass screen for machine fit planning. Use it to identify the next record or engineering check, not as a release decision.

Sanity check

Check the unit basis and whether all quantities describe the same cavity, shot, machine, or production window before comparing values.

Important limitation

Daylight, ejector coupling, platen travel, and installation clearances are separate checks.

Reference: ENGEL: machine data example.
