Scrap-Adjusted Output Calculator

Convert gross molded output and scrap rate into expected good output. Use this as a documented planning estimate, then validate it against the actual machine, mold, resin, and process.

Formula and method

Good output = gross output x (1 - scrap rate).

Use gross and scrap values measured over the same production population.

Inputs and units

Use controlled records, CAD, measured production data, the exact resin data sheet, or machine-maker specifications. Keep the material and time basis with the result.

Worked example

10,000 gross parts at 3% scrap = 9,700 good parts.

What the result supports

This result is a first-pass screen for quality planning. Use it to identify the next record or engineering check, not as a release decision.

Sanity check

Check the unit basis and whether all quantities describe the same cavity, shot, machine, or production window before comparing values.

Important limitation

This does not diagnose a defect mechanism or account for rework policy.

Reference: ARBURG: cycle-time optimisation.
