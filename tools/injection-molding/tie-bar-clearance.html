Tie-Bar Clearance Calculator

Check the remaining width and height clearance inside a four-tie-bar opening. Use this as a documented planning estimate, then validate it against the actual machine, mold, resin, and process.

Formula and method

Clearance = tie-bar opening minus mold envelope.

Use the machine’s published tie-bar spacing, not the platen size; allow for handling and installed components.

Inputs and units

Use controlled records, CAD, measured production data, the exact resin data sheet, or machine-maker specifications. Keep the material and time basis with the result.

Worked example

An 800 x 700 mm mold in a 900 x 800 mm opening leaves 100 mm each way.

What the result supports

This result is a first-pass screen for machine fit planning. Use it to identify the next record or engineering check, not as a release decision.

Sanity check

Check the unit basis and whether all quantities describe the same cavity, shot, machine, or production window before comparing values.

Important limitation

Tie-bar-less machines and core pulls need their own equipment review.
